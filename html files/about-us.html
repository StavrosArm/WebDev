Αρμενιάκος Σταύρος

Είμαι φοιτητής στο τμήμα Πληροφορικής του Οικονομικού Πανεπιστημίου Αθηνών. Η αγάπη μου για τον κόσμο της πληροφορικής με καθοδηγεί στην ανακάλυψη συναρπαστικών πτυχών της τεχνολογίας. Μέσα από τις σπουδές μου αλλά και την εκπλήρωση διαφόρων project, αναζητώ συνεχώς νέες προκλήσεις που θα εμπλουτίσουν την κατανόησή μου για τον κόσμο του προγραμματισμού. Η διαρκής εξέλιξη της τεχνολογίας αναζωογονεί το πάθος μου για αυτήν και με καλεί να συμβάλλω στην δημιουργία καινοτόμων λύσεων.Τα κύρια ενδιαφέροντα μου αφορούν την Μηχανική Μάθηση, το Web Development καθώς και το Data Science. Ακολουθήστε με στo StavrosArm στο GitHub για να δείτε το portfolio μου.

Ιωάννου Κωσταντίνος

Είμαι ένας φοιτητής στο Οικονομικό Πανεπιστήμιο Αθηνών, όπου φοιτώ στο τμήμα πληροφορικής. Βρίσκομαι στο 7ο έτος των σπουδών μου, μια πορεία που με έχει εμπνεύσει και με έχει βοηθήσει να εξελίξω τις γνώσεις μου στον κόσμο της πληροφορικής. Ένας τομέας που με έχει ενθουσιάσει ιδιαίτερα είναι το Web Development. Από τη δημιουργία εντυπωσιακών ιστοσελίδων μέχρι την ανάπτυξη λειτουργικών εφαρμογών, το Web Development με έχει εντυπωσιάσει με τις ατελείωτες δυνατότητες που προσφέρει. Επίσης, μπορείτε να βρείτε τις δουλειές μου και τα projects που έχω αναλάβει στο GitHub, στον παρακάτω σύνδεσμο: Kostantinos-Ioannou . Ανυπομονώ να συνεχίσω να εξερευνώ τον υπέροχο κόσμο της πληροφορικής και να συνεχίσω να αναπτύσσω τις δεξιότητές μου στο Web Development.
